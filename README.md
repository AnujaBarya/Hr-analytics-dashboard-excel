# HR Analytics Dashboard

## Project Overview
This project showcases an interactive HR Analytics Dashboard built using Tableau.  
The dashboard provides insights into employee attrition, department performance, workforce demographics, job satisfaction, and employee distribution trends.

---

## Tools Used
- Tableau
- Excel
- Data Visualization
- Dashboard Design

---

## Dashboard Features
- Employee Count Analysis
- Attrition Count & Attrition Rate
- Active Employee Tracking
- Average Employee Age Analysis
- Department-wise Attrition
- Employee Age Group Distribution
- Job Satisfaction Rating Analysis
- Education Field-wise Attrition
- Gender-wise Attrition Analysis

---

## Key Insights
- Total employee count reached 1,470.
- Attrition count recorded was 237 with an attrition rate of 16.12%.
- Research & Development department showed the highest attrition.
- Employees aged 25–34 had the highest attrition rate.
- Sales Executives and Research Scientists had high employee representation.
